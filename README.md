# 🧮 Simple Calculator

A modern, lightweight web-based calculator with a beautiful UI and smooth interactions.

## ✨ Features

- ✅ **Basic Arithmetic Operations** - Addition, Subtraction, Multiplication, Division
- ✅ **Modulo Operation** - Calculate remainders
- ✅ **Decimal Support** - Work with decimal numbers
- ✅ **Delete Function** - Remove the last digit with the delete button
- ✅ **Clear Function** - Reset the calculator instantly
- ✅ **Error Handling** - Graceful error management
- ✅ **Responsive Design** - Works on all devices
- ✅ **Modern UI** - Beautiful gradient background and smooth animations
- ✅ **Keyboard Friendly** - Easy-to-use button interface

## 🎨 Design Highlights

- **Beautiful Gradient Background** - Purple gradient for modern aesthetics

## 🚀 How to Use

1. **Clone the Repository**
   ```bash
   git clone git@github.com:Salto123-dev/simple-calculator.git
   cd simple-calculator
   ```

2. **Open the Calculator**
   - Open index.html in your web browser
   - Or use a local server

3. **Perform Calculations**
   - Click number buttons to enter values
   - Click operator buttons for operations
   - Click = to calculate
   - Click C to clear
   - Click ← to delete the last digit
